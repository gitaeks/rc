Designed for everyone, everywhere.

Ratchet Plus makes front-end mobile web and app development faster and easier. It's made for folks of all skill levels, devices of all shapes, and projects of all sizes.

Preprocessors

Ratchet Plus ships with vanilla CSS, but its source code utilizes the two most popular CSS preprocessors, Less and Sass. Quickly get started with precompiled CSS or build on the source.

One framework, every device.

Ratchet Plus easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

Full of features

With Ratchet Plus, you get extensive and beautiful documentation for common HTML elements, dozens of custom HTML and CSS components, and awesome jQuery plugins.

Ratchet Plus is open source. It's hosted, developed, and maintained on GitHub.
